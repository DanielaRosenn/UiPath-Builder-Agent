# Maintenance Scripts

Utilities in this folder support verification, reporting, or local maintenance.

## Rules

- Scripts here are not required by runtime chat flow.
- Keep dependencies minimal and documented in each script header.
- Prefer deterministic output and explicit exit codes for CI usage.
