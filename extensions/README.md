# Extensions

Git-tracked **team** material that must not live in the `skills/` submodule (UiPath upstream) or under personal-only paths (`~/.cursor/skills`).

| Subfolder | Purpose |
| --- | --- |
| [skills/](skills/README.md) | Skill overlays merged before the submodule (see `docs/SKILL_LAYOUT.md`). |
| [skill-insights/](skill-insights/README.md) | Curated insight JSON promoted via PR. |
| [uipath-rule-bundle/](uipath-rule-bundle/README.md) | Portable UiPath AI rule bundle for **other** repos (copy-out), not imported by the Python framework here. |

For merge order and Cursor vs CLI behavior, see [docs/SKILL_LAYOUT.md](../docs/SKILL_LAYOUT.md).
